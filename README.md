# Usage
```
usage: scrap.py [-h] [--json] url

positional arguments:
  url         The URL to check

options:
  -h, --help  show this help message and exit
  --json      Output the result in JSON format
```
# Example
```
Normmal Usage : python3 statusFinder.py google.com
json format  : python3 statusFinder.py google.com --json
```
